# 管理后台布局说明

## ✅ 布局结构

管理后台采用**传统左侧导航栏布局**，结构如下：

```
┌─────────────────────────────────────────┐
│                                         │
│  ┌──────────┐  ┌────────────────────┐  │
│  │          │  │                    │  │
│  │  侧边栏  │  │   顶部导航栏       │  │
│  │  (固定)  │  ├────────────────────┤  │
│  │          │  │                    │  │
│  │ 导航菜单 │  │   页面内容区       │  │
│  │          │  │                    │  │
│  │          │  │                    │  │
│  └──────────┘  └────────────────────┘  │
│                                         │
└─────────────────────────────────────────┘
```

## 📋 布局组成

### 1. 左侧侧边栏（固定）
- **位置**: 页面左侧固定
- **宽度**: 250px
- **背景**: 渐变蓝色（#1a237e → #283593）
- **高度**: 100vh（全屏高度）
- **包含内容**:
  - Logo 区域（小蚁搬运 + 总后台管理系统）
  - 导航菜单

### 2. 导航菜单项
- ✅ **仪表盘** - 数据统计和概览
- ✅ **订单管理** - 订单列表和管理
- ✅ **租户管理** - 租户列表和管理
- ✅ **接单人员管理** - 工人管理
- ✅ **用户管理** - 用户列表和管理
- ✅ **报表统计** - 数据报表和图表
- ✅ **系统设置** - 系统配置

### 3. 顶部导航栏
- **位置**: 主内容区顶部
- **高度**: 60px
- **背景**: 白色
- **包含内容**:
  - 侧边栏切换按钮（移动端）
  - 搜索框
  - 用户下拉菜单（个人资料、设置、退出登录）

### 4. 主内容区
- **位置**: 侧边栏右侧
- **左边距**: 250px（侧边栏宽度）
- **背景**: 浅灰色（#f5f5f5）
- **包含**: 各个页面的内容

## 🎨 CSS 配置

### 侧边栏样式
```css
.sidebar {
  width: 250px;
  background: linear-gradient(180deg, #1a237e, #283593);
  color: white;
  position: fixed;
  top: 0;
  left: 0;
  height: 100vh;
  z-index: 1000;
}
```

### 主面板样式
```css
.main-panel {
  flex: 1;
  margin-left: 250px; /* 侧边栏宽度 */
}
```

### 响应式设计
- **桌面端**（>768px）: 侧边栏固定显示
- **移动端**（≤768px）: 侧边栏隐藏，通过按钮切换显示

## 📱 响应式特性

### 桌面端
- 侧边栏始终显示在左侧
- 主内容区自动调整宽度
- 顶部导航栏显示完整功能

### 移动端
- 侧边栏默认隐藏
- 点击汉堡菜单按钮显示侧边栏
- 主内容区占满全屏
